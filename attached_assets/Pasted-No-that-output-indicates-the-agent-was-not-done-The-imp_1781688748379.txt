### What's bringing this error?

The key message is:

```text
invalid internal format (DSML tool-call syntax)
```

This usually means the model generated a malformed tool invocation. Common causes:

* The model output exceeded complexity limits.
* The model mixed normal text and tool-call syntax incorrectly.
* The orchestrator expected JSON/tool format but received partially formatted output.
* A coding model tried to call tools using the wrong schema.
* Context became too large (your logs show ~35k tokens).
