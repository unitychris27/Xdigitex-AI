## 9. Your Biggest Missing Component

From all the failures you showed, the missing component isn't another model.

It's a **Verifier Controller**.

Something like:

```python
class Verifier:
    def website_loaded(self): ...
    def login_worked(self): ...
    def db_connected(self): ...
    def page_not_blank(self): ...
    def screenshot_matches(self): ...
```

Every task must pass verifier checks before the agent can continue.

That would have prevented:

* "Registration successful" when it wasn't.
* "Login successful" when it wasn't.
* Blank-page success claims.
* Declaring sites complete without checking routes.
* Claiming deployment worked without HTTP verification.
