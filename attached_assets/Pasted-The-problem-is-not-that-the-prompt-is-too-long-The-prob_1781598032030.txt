## Another Improvement

Add this rule:

```text
Never create placeholder code.

Never create TODO comments.

Never create stubs.

Every file must be fully functional.
```

Otherwise agents generate things like:

```php
// TODO: Add Stripe Integration
```

and claim the feature is complete.

For a project the size of LinkForge, the highest success rate is usually:

**Database → Core Framework → Auth → Dashboard → Links → Analytics → Payments → Admin → Polish**

rather than trying to generate the entire SaaS platform in a single prompt. That keeps the AI focused and greatly increases the chance that each module is actually completed and tested.
