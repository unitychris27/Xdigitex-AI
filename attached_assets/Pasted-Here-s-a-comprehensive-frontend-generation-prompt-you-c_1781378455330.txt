# Frontend System Prompt

Build a modern SaaS frontend for an AI-powered Multi-Agent Development Platform.

The platform allows users to build applications, deploy Telegram bots, manage AI agents, connect servers, store secrets securely, monitor deployments, manage billing, referrals, promotions, and administration.

Design should feel like a combination of:

* GitHub
* Cursor
* Vercel
* Linear
* Railway
* Stripe Dashboard

The UI must be professional, developer-focused, enterprise-grade, responsive, and production-ready.

---

## Design Requirements

Create:

* Modern dark mode by default
* Light mode support
* Responsive desktop/tablet/mobile
* Clean spacing
* Premium SaaS appearance
* Smooth animations
* Skeleton loaders
* Empty states
* Error states
* Notification system
* Global search
* Keyboard shortcuts

Use:

* React
* Next.js
* TypeScript
* TailwindCSS
* shadcn/ui style components
* Framer Motion
